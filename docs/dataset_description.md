# Dataset Description

## Overview

EvidenceRank AI was developed using a recruitment dataset containing 100,000 anonymized candidate profiles.

The dataset was provided as part of an AI candidate ranking challenge and includes structured candidate information, career history, skills, certifications, education, recruiter signals, and behavioral indicators.

## Dataset Size

- Total Candidates: 100,000
- File Format: JSONL
- Dataset Size: ~487 MB

## Available Candidate Information

### Profile Information

- Candidate ID
- Current Title
- Current Company
- Industry
- Experience
- Location
- Professional Summary

### Career History

- Previous Roles
- Employment Duration
- Company Information
- Project Descriptions

### Skills

- Skill Name
- Proficiency Level
- Endorsements
- Experience Duration

### Education

- Degree
- Institution
- Field of Study
- Academic Performance

### Certifications

- Professional Certifications

### Behavioral Signals

- Open To Work Status
- Recruiter Response Rate
- Notice Period
- Profile Activity
- Interview Completion Rate
- Offer Acceptance Rate
- Saved By Recruiters
- Profile Completeness

## Data Privacy

The original dataset is not distributed in this repository due to challenge restrictions and licensing requirements.
